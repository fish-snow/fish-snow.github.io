6 void PatchDword(long addr, long val) 设置虚拟地址addr处的一个双字值

操纵数据库名称函数

序号 函数名及参数与返回值 作用
1 string Name(long addr) 返回与地址有关的名称，如果该位置没有名称，则返回空字符串。如果名称被标记为局部名称，这个函数并不返回用户定义的名称
2 string NameEx(long from, long addr) 返回与addr有关的名称。如果该位置没有名称，则返回空字符串
3 bool MakeNameEx(long addr, string name, long flags)
4 long LocByName(string name) 返回一个位置（名称已给定）的地址。如果数据库中没有这个名称，则返回BADADDR(-1)
5 long LocByNameEx(long funcaddr, string localname) 在包含funcaddr的函数中搜索给定的局部名称。如果给定的函数中没有这个名称,则返回BADADDR(-1)

处理函数的函数

序号 函数名，参数及返回值 作用
1 long GetFunctionAttr(long addr, long attrib) 返回包含给定地址的函数的被请求的属性(属性见下文)
2 string GetFunctionName(long addr) 返回包含给定地址的函数名称。如果给定的地址并不属于一个函数，则返回一个空字符串
3 long NextFunction(long addr) 返回给定地址之前距离最近的函数的起始地址。如果在给定地址后没有其他函数，则返回-1
4 long PrevFunction(long addr) 返回给定地址之前距离最近的函数起始地址。如果在给定地址之前没有函数，则返回-1
