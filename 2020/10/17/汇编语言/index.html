标志寄存器

8086cupflag寄存器示意图如下，其中1,3,5,12,13,14,15位没有使用。

标志 作用简介
ZF 记录相关指令 执行后结果是否为0
PF 记录相关指令执行后，其结果的所有Bit位中1的个数是否为偶数
SF 记录相关指令执行后，其结果是否为负
CF 在无符号运算时，记录运算结果的最高位向更高位的进位值，或从更高位的借位值
OF 记录相关指令执行后，是否溢出
DF

CF标志示例

CF为进位标志位。一般情况下，在进行无符号数运算的时候，它记录了运算结果的最高有效位向更高位的进位值，或从更高位的借位值

sbb指令

带进位的减法

指令格式：sbb reg1,reg2 或者 sbb reg ,立即数

实际操作：reg1 - reg2 - CF

abc指令

带进位的加法

指令格式：abc reg1,reg2 或者 abc reg,立即数

实际操作：reg1 + reg2 + 2

cmp指令

cmp指令会影响标志寄存器，后常跟JCC指令，下面为JCC指令表

JCC指令表

指令助记符 检查标记位 说明
JZ ZF == 1 等于0则跳转
JE ZF == 1 相等则跳转
JNZ ZF == 0 不等于0则跳转
JNE ZF == 0 不相等则跳转
JS SF == 1 符号为负则跳转
JNS SF == 0 符号为正则跳转
JP/JPE PF == 1 1 的个数为偶数则跳转
JNP/JPO PF == 0 1 的个数为奇数则跳转
JO OF == 1 溢出则跳转
JNO OF == 0 无溢出则跳转
JC CF == 1 进位则跳转
JB CF == 1 小于则跳转
JNAE CF == 1 不大于等于则跳转
JNC CF == 0 无进位则跳转
JNB CF == 0 不小于则跳转
JAE CF == 0 大于则跳转
JBE CF == 1 || ZF == 1 小于等于则跳转
JNA CF == 1 || ZF == 1 不大于则跳转
JNBE CF == 0 || ZF == 0 不小于等于则跳转
JA CF == 0 || ZF == 0 大于则跳转
JL SF != OF 小于则跳转
JNGE SF != OF 不大于等于则跳转
JNL SF == 0F 不小于则跳转
JGE SF == OF 不大于等于则跳转
JLE ZF != OF || ZF == 1 小于等于则跳转
JNG ZF != OF || ZF ==1 不大于则跳转
JNLE SF == OF && ZF == 0 不小于则跳转
JG SF == OF && ZF ==0 大于则跳转

setg指令

指令格式：setg al
当标记为位SF=OF 且 ZF=0 赋值al为1

neg指令

指令格式：neg reg
reg为寄存器名字
作用：对reg中存储的数据取反

shl指令

左移运算指令

指令格式：shl 操作数1，移动位数

sar指令(有符号运算)

右移运算指令，但是会保留最高位(符号位)

指令格式：sar 操作数1，移动位数

shl指令(无符号运算)

右移运算指令，不会保留最高位

not指令

取反运算

指令格式：not reg
